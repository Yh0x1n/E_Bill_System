## Prepare Environment

Isnstall virtual env
```
pip install virtualenv
```

Create the environment 
```
python -m venv myenv
```

On windows run: 
```
myenv\Scripts\activate
```


## Install dependencies 

Upgrade pip if its needed
```
python.exe -m pip install --upgrade pip
```

Install everything
```
pip install -r requirements.txt
```

Install pyqt
```
pip install pyside6
```

pandas
```
pip install pandas
```

## Run the app
```
python .
```
